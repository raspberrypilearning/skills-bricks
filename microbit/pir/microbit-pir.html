PIR Sensor

Detect Motion

What does it do?

A PIR sensor tells your project when something warm moves nearby.

PIR stands for passive infrared. It detects changes in infrared energy, so it is good at noticing people moving.

With a micro:bit, you read the sensor's output on a digital pin.

JavaScript

basic.forever(function () {
    if (pins.digitalReadPin(DigitalPin.P0) == 1) {
        basic.showIcon(IconNames.Yes)
    }
})
pin0.read_digital() is 1 when movement is detected. Many PIR modules need 5V power and a short warm-up after switching on.
