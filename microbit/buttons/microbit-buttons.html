Detect Press

What does it do?

The micro:bit has two buttons, A and B, that your code can react to when they are pressed.

They are built into the board, so there is nothing to wire up. Button A is on the left and Button B is on the right.

Your projects can use the buttons to choose options, control characters, or trigger actions.

MicroPython

from microbit import *

while True:
    if button_a.was_pressed():
        display.show("A")
    if button_b.was_pressed():
        display.show("B")

JavaScript

input.onButtonPressed(Button.A, function () {
    basic.showString("A")
})
input.onButtonPressed(Button.B, function () {
    basic.showString("B")
})
Use is_pressed() to check if a button is held down right now, or was_pressed() to catch a press that already happened.

Quick reference

button_a, button_bThe two named buttons
is_pressed()True while the button is held down
was_pressed()True once after a press
Built inNo wiring needed — both buttons are on the board
