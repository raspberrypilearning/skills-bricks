micro:bit SPI

Connect Fast Data

What does it do?

SPI is a fast way for your micro:bit to send data to modules.

It uses a clock wire, data wires, and usually a chip select pin.

Use SPI with displays, memory modules, LED drivers, and other speedy parts.
