What does it do?

A toggle or slide switch stays in the position you leave it — on or off.

Unlike a button, it does not spring back, so it remembers its setting.

With a micro:bit, you read it on a pin to choose a mode or turn things on and off.

MicroPython

from microbit import *

pin0.set_pull(pin0.PULL_UP)
while True:
    if pin0.read_digital() == 1:
        display.show("1")
    else:
        display.show("0")

JavaScript

pins.setPull(DigitalPin.P0, PinPullMode.PullUp)
basic.forever(function () {
    if (pins.digitalReadPin(DigitalPin.P0) == 1) {
        basic.showString("1")
    } else {
        basic.showString("0")
    }
})
A toggle switch holds its state, so the pin stays HIGH or LOW until you flip it. Set a pull-up and read it as digital.
