RGB LED

Output Colour

What does it do?

An RGB LED is three coloured lights — red, green and blue — built into one part.

By mixing how bright each colour is, your code can make almost any colour.

With a micro:bit, you connect the three colour legs to the three PWM pins and set each level.

MicroPython

from microbit import *

pin0.write_analog(1023)  # red
pin1.write_analog(400)   # green
pin2.write_analog(0)     # blue

JavaScript

pins.analogWritePin(AnalogPin.P0, 1023)
pins.analogWritePin(AnalogPin.P1, 400)
pins.analogWritePin(AnalogPin.P2, 0)
Each colour uses a PWM pin (P0, P1, P2) — that is all three of the micro:bit's analog outputs. Add a resistor on each leg; for a common-anode LED, pin0.write_analog() with 0 is brightest.
