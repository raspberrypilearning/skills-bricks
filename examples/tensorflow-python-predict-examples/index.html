TensorFlow Python Predict Examples

Two Python examples for a Teachable Machine image model: one quick image check and one webcam object quest.

Train First

Use a Teachable Machine Image Project with classes named pencil, cup, book, and background.

Export as TensorFlow/Keras if available, or TensorFlow Lite if that is the only Python-friendly option.

Still Image

Run python3 predict_image.py sample_images/pencil-test.jpg to print the top prediction.

Webcam Game

Run python3 object_quest_game.py and show the right object to the camera before time runs out.
