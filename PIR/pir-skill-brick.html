PIR

Detect Motion

What does it do?

A PIR sensor helps your project notice movement from people or warm things nearby.

PIR stands for passive infrared. It watches for changes in heat moving across its view.

Your projects can use PIR sensors for lights, alarms, counters, and dramatic entrances.

Code

Raspberry Pi

from gpiozero import MotionSensor

pir = MotionSensor(17)

motion = pir.motion_detected

Raspberry Pi Pico

from picozero import MotionSensor

pir = MotionSensor(2)

motion = pir.motion_detected
Use motion when your project needs to react to movement.
