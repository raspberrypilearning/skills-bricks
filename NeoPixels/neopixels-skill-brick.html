NeoPixels

Do Pixels

What does it do?

NeoPixels are colourful lights that can be controlled one by one from a single data wire.

Each pixel remembers its own colour, so a strip can become a sign, meter, clock, or rainbow trail.

Your projects can use NeoPixels for badges, reactions, animations, and unnecessary sparkle.

Code

Raspberry Pi

from rpi_ws281x import PixelStrip, Color

pixels = PixelStrip(8, 18)
pixels.begin()

pixels.setPixelColor(0, Color(255, 0, 80))
pixels.show()

Raspberry Pi Pico

from machine import Pin
from neopixel import NeoPixel

pixels = NeoPixel(Pin(2), 8)

pixels[0] = (255, 0, 80)
pixels.write()
Use pixel colours later in your Do code.
