Button

Input Press

What does it do?

A button lets your project notice when someone presses something.

Inside the button are tiny metal contacts. Pressing the cap joins the contacts together, closing the circuit.

Your projects can use a button to start actions, choose modes, confirm choices, or act like a simple controller.

Code

Raspberry Pi

from gpiozero import Button

button = Button(17)

pressed = button.is_pressed

Raspberry Pi Pico

from picozero import Button

button = Button(2)

pressed = button.is_pressed
Use button.is_pressed or button.when_pressed when your project needs to react to a press.
