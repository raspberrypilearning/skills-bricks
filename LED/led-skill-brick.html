LED

Output Light

What does it do?

An LED is a tiny light that your code can switch on, switch off, or blink.

LED stands for light-emitting diode. It glows when electricity flows the right way through it, so the long leg and short leg matter.

Your projects can use LEDs to show warnings, celebrate events, or tell people what is happening.

Code

Raspberry Pi

from gpiozero import LED

led = LED(17)

led.on()

Raspberry Pi Pico

from picozero import LED

led = LED(2)

led.on()
Use led.on(), led.off(), or led.blink() later in your Do code.
