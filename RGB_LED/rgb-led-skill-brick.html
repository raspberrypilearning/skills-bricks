RGB LED

Do Colour

What does it do?

An RGB LED combines red, green, and blue lights in one part.

By changing the brightness of each colour, your code can mix lots of different colours.

Your projects can use RGB LEDs for moods, warnings, game states, and small light shows.

Code

Raspberry Pi

from gpiozero import RGBLED

led = RGBLED(red=17, green=27, blue=22)

led.color = (1, 0, 1)

Raspberry Pi Pico

from picozero import RGBLED

led = RGBLED(red=2, green=3, blue=4)

led.color = (1, 0, 1)
Use led.color later in your Do code.
