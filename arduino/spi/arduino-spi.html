SPI

Connect Fast Data

What does it do?

SPI is a fast way to connect modules like screens, card readers and radios.

It uses separate data-in, data-out and clock lines, plus a select pin for each device.

With an Arduino, the SPI library handles the timing for you.

Code

Arduino

#include <SPI.h>

void setup() {
  SPI.begin();
  pinMode(SS, OUTPUT);
}

void loop() {
  digitalWrite(SS, LOW);
  SPI.transfer(0x42);
  digitalWrite(SS, HIGH);
}
SPI uses pins 13 (SCK), 12 (MISO) and 11 (MOSI) on an Uno, plus a select pin for each device.
