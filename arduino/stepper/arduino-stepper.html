Stepper Motor

Do Motion

What does it do?

A stepper motor moves in small exact steps instead of spinning freely.

This lets your project turn something to a precise position, like a dial or a pointer.

With an Arduino, the Stepper library drives it through a board such as a ULN2003.

Code

Arduino

#include <Stepper.h>

Stepper motor(2048, 8, 10, 9, 11);

void setup() {
  motor.setSpeed(10);
}

void loop() {
  motor.step(2048);
  delay(1000);
}
A 28BYJ-48 motor has 2048 steps per turn. The ULN2003 driver board needs its own power supply.
