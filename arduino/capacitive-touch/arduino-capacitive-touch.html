Capacitive Touch

Detect Touch

What does it do?

A capacitive touch sensor reacts to your finger, just like a phone screen.

It senses the tiny electrical change your skin makes when you touch the pad.

With an Arduino, a touch module gives a digital HIGH when it is touched.

Code

Arduino

const int touch = 2;

void setup() {
  pinMode(touch, INPUT);
  Serial.begin(9600);
}

void loop() {
  if (digitalRead(touch) == HIGH) {
    Serial.println("Touched");
  }
}
A TTP223 touch module outputs HIGH while the pad is touched — read it like a button.
