What does it do?

An LED is a tiny light that your code can switch on, switch off, or blink.

LED stands for light-emitting diode. It glows when electricity flows the right way through it, so the long leg and short leg matter.

With an Arduino, you connect an LED to a pin through a resistor and switch it on and off in your sketch.

Code

Arduino

const int led = 13;

void setup() {
  pinMode(led, OUTPUT);
}

void loop() {
  digitalWrite(led, HIGH);
  delay(500);
  digitalWrite(led, LOW);
  delay(500);
}
Use digitalWrite(led, HIGH) to turn the LED on and LOW to turn it off. Always put a 220 Ω resistor in series with the LED.
