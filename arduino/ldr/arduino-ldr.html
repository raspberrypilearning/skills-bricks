LDR

Detect Level

What does it do?

An LDR is a light sensor that tells your project how bright or dark it is.

LDR stands for light dependent resistor. Its resistance changes when more or less light falls on the sensor face.

With an Arduino, you read the LDR on an analog pin to get a light level.

Code

Arduino

const int ldr = A0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int light = analogRead(ldr);
  Serial.println(light);
}
analogRead() returns 0–1023. Wire the LDR with a 10kΩ resistor as a voltage divider into A0.
