Motor Driver

Do Motor

What does it do?

A motor driver board lets your Arduino control motors that need more power than a pin can supply.

The Arduino sends low-power signals and the driver switches the larger motor current from its own supply.

You set direction with digital pins and speed with a PWM pin.

Code

Arduino

const int in1 = 5;
const int in2 = 6;
const int ena = 9;

void setup() {
  pinMode(in1, OUTPUT);
  pinMode(in2, OUTPUT);
  pinMode(ena, OUTPUT);
}

void loop() {
  digitalWrite(in1, HIGH);
  digitalWrite(in2, LOW);
  analogWrite(ena, 180);
}
Power the motors from the driver board's own battery, not the Arduino. analogWrite() on ENA sets the speed.
