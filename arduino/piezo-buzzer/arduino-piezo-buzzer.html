Piezo Buzzer

Do Sound

What does it do?

A piezo buzzer is a small part that makes a buzz or beep when your code turns it on.

It turns electrical energy into vibration, which makes a sound you can hear.

With an Arduino, you play tones through a pin using the tone() function.

Code

Arduino

const int buzzer = 8;

void setup() {
  pinMode(buzzer, OUTPUT);
}

void loop() {
  tone(buzzer, 440, 500);
  delay(1000);
}
tone(pin, frequency, duration) plays a note. Use noTone() to stop it early.
