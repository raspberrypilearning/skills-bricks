Temperature

What does it do?

A temperature sensor lets your project measure how warm or cold it is.

An analog sensor like the TMP36 gives a voltage that rises with temperature.

With an Arduino, you read it on an analog pin and convert the value to °C.

Code

Arduino

const int tmp = A0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int reading = analogRead(tmp);
  float volts = reading * 5.0 / 1024.0;
  float tempC = (volts - 0.5) * 100;
  Serial.println(tempC);
}
A TMP36 gives 10 mV per °C with a 0.5V offset. Convert the analog reading to volts, then to °C.
