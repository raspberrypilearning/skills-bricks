Potentiometer

Detect Level

What does it do?

A potentiometer is a knob that changes its resistance as you turn it.

Wired as a voltage divider, it gives the Arduino a value that rises and falls as you turn it.

With an Arduino, you read it on an analog pin to get a number from 0 to 1023.

Code

Arduino

const int pot = A0;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int value = analogRead(pot);
  Serial.println(value);
}
analogRead() gives 0 at one end and 1023 at the other.
