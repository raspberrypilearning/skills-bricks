Ultrasonic Distance Sensor

Detect Level

What does it do?

A distance sensor measures how far away something is without touching it.

It works a bit like a bat, sending out sound and timing the echo that bounces back.

With an Arduino, you send a pulse on one pin and time the echo on another.

Code

Arduino

const int trig = 9;
const int echo = 10;

void setup() {
  pinMode(trig, OUTPUT);
  pinMode(echo, INPUT);
  Serial.begin(9600);
}

void loop() {
  digitalWrite(trig, LOW);
  delayMicroseconds(2);
  digitalWrite(trig, HIGH);
  delayMicroseconds(10);
  digitalWrite(trig, LOW);
  long us = pulseIn(echo, HIGH);
  float cm = us * 0.0343 / 2;
  Serial.println(cm);
}
pulseIn() times the echo; multiply by the speed of sound to get the distance in cm.
