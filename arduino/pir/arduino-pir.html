PIR Sensor

Detect Motion

What does it do?

A PIR sensor tells your project when something warm moves nearby.

PIR stands for passive infrared. It detects changes in infrared energy, so it is good at noticing people moving.

With an Arduino, you read the sensor's output on a digital pin.

Code

Arduino

const int pir = 2;

void setup() {
  pinMode(pir, INPUT);
  Serial.begin(9600);
}

void loop() {
  if (digitalRead(pir) == HIGH) {
    Serial.println("Motion!");
  }
}
digitalRead() is HIGH when movement is detected. Most PIR modules run on 5V and need a short warm-up.
