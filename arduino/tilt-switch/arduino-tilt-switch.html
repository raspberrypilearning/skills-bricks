Tilt Switch

Detect Motion

What does it do?

A tilt switch is a small switch that turns on or off when it is tipped.

Inside is usually a tiny ball that rolls to make or break the connection.

With an Arduino, you read it on a digital pin, just like a button.

Code

Arduino

const int tilt = 2;

void setup() {
  pinMode(tilt, INPUT_PULLUP);
  Serial.begin(9600);
}

void loop() {
  if (digitalRead(tilt) == LOW) {
    Serial.println("Tilted");
  }
}
With INPUT_PULLUP the pin reads HIGH normally and LOW when the ball closes the contacts. Read it like a button.
