Speaker

Do Sound

What does it do?

A speaker turns electrical signals into sound you can hear.

On an Arduino you can drive a small speaker with the tone() function to play notes and tunes.

A speaker can play richer sound than a simple buzzer.

Code

Arduino

const int speaker = 8;
int tune[] = {262, 294, 330, 349};

void setup() {
  pinMode(speaker, OUTPUT);
}

void loop() {
  for (int i = 0; i < 4; i++) {
    tone(speaker, tune[i], 300);
    delay(350);
  }
}
tone() plays square-wave notes through the speaker. For louder, cleaner sound, drive it through a small transistor or amplifier.
