Button / Switch

Detect Press

What does it do?

A button or switch lets your project take input when someone presses or flips it.

When pressed it connects a pin to GND, which your code reads as a digital value.

With an Arduino, you use the internal pull-up resistor and read the pin as digital.

Code

Arduino

const int button = 2;

void setup() {
  pinMode(button, INPUT_PULLUP);
  Serial.begin(9600);
}

void loop() {
  if (digitalRead(button) == LOW) {
    Serial.println("Pressed");
  }
}
With INPUT_PULLUP the pin reads HIGH normally and LOW when pressed (button wired to GND).
