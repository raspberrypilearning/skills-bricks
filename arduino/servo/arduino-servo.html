Servo

Do Motion

What does it do?

A servo motor turns to an exact angle and holds it there.

It is great for arms, gates, dials, and steering.

With an Arduino, the Servo library moves it to an angle between 0 and 180 degrees.

Code

Arduino

#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(9);
}

void loop() {
  myServo.write(0);
  delay(1000);
  myServo.write(90);
  delay(1000);
}
Include the Servo library. write() takes an angle from 0 to 180 degrees.
