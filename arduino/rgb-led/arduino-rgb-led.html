RGB LED

Do Colour

What does it do?

An RGB LED is three coloured lights — red, green and blue — built into one part.

By mixing how bright each colour is, your code can make almost any colour.

With an Arduino, you connect the three colour legs to PWM pins and set each level.

Code

Arduino

const int r = 9;
const int g = 10;
const int b = 11;

void setup() {
  pinMode(r, OUTPUT);
  pinMode(g, OUTPUT);
  pinMode(b, OUTPUT);
}

void loop() {
  analogWrite(r, 255);
  analogWrite(g, 100);
  analogWrite(b, 0);
}
Use PWM pins (marked ~) and analogWrite() 0–255 per colour. Add a resistor on each leg.
