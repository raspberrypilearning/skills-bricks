Magnetometer

Detect Direction

What does it do?

A magnetometer measures magnetic fields, which lets the Arduino work as a compass.

It can give a heading in degrees where 0 means north.

With an Arduino, a compass module is read over I²C using a library.

Code

Arduino

#include <Wire.h>
#include <QMC5883LCompass.h>

QMC5883LCompass compass;

void setup() {
  Serial.begin(9600);
  compass.init();
}

void loop() {
  compass.read();
  Serial.println(compass.getAzimuth());
}
Install a compass library. The module connects over I²C (SDA/SCL); getAzimuth() returns the heading.
