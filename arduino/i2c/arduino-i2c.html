I2C

Connect Data

What does it do?

I²C is a way to connect modules like displays and sensors using just two shared wires.

Many devices can share the same two lines, each with its own address.

With an Arduino, the Wire library talks to I²C devices on A4 and A5.

Code

Arduino

#include <Wire.h>

void setup() {
  Wire.begin();
  Serial.begin(9600);
}

void loop() {
  Wire.requestFrom(0x68, 1);
  if (Wire.available()) {
    int data = Wire.read();
    Serial.println(data);
  }
}
The Wire library uses A4 (SDA) and A5 (SCL) on an Uno. Each device has its own address.
