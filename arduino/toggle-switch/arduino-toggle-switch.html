Toggle Switch

Detect State

What does it do?

A toggle or slide switch stays in the position you leave it — on or off.

Unlike a button, it does not spring back, so it remembers its setting.

With an Arduino, you read it on a digital pin to choose a mode or turn things on and off.

Code

Arduino

const int sw = 2;

void setup() {
  pinMode(sw, INPUT_PULLUP);
  Serial.begin(9600);
}

void loop() {
  if (digitalRead(sw) == LOW) {
    Serial.println("ON");
  } else {
    Serial.println("OFF");
  }
}
A toggle switch holds its state, so the pin stays HIGH or LOW until you flip it. INPUT_PULLUP gives a clean reading.
