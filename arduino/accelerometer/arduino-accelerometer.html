Accelerometer

Detect Motion

What does it do?

An accelerometer senses movement, tilt, and how it is being held.

It measures force on three axes — x, y and z.

With an Arduino, an analog accelerometer is read on three analog pins.

Code

Arduino

const int xPin = A0;
const int yPin = A1;
const int zPin = A2;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int x = analogRead(xPin);
  int y = analogRead(yPin);
  int z = analogRead(zPin);
  Serial.println(x);
}
An analog accelerometer (like the ADXL335) gives a voltage per axis that you read with analogRead().
