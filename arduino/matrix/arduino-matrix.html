LED Matrix

Do Light

What does it do?

An LED matrix is a grid of LEDs you can light up to show numbers, letters and pictures.

A common module is an 8×8 grid driven by a MAX7219 chip.

With an Arduino, the LedControl library lights any LED in the grid.

Code

Arduino

#include <LedControl.h>

LedControl lc(12, 11, 10, 1);

void setup() {
  lc.shutdown(0, false);
  lc.setIntensity(0, 8);
  lc.clearDisplay(0);
}

void loop() {
  lc.setLed(0, 2, 3, true);
}
Install the LedControl library. lc(DIN, CLK, CS, devices); setLed() lights one LED by row and column.
