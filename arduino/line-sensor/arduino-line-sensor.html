Line Sensor

Detect Surface

What does it do?

A line sensor shines infrared light down and measures how much bounces back.

A dark line reflects little light and a pale surface reflects a lot, so the sensor can tell them apart.

With an Arduino, you read the sensor on a pin to follow lines or spot edges.

Code

Arduino

const int line = 2;

void setup() {
  pinMode(line, INPUT);
  Serial.begin(9600);
}

void loop() {
  int onLine = digitalRead(line);
  Serial.println(onLine);
}
Most modules output digital (1 and 0 for line and no line). Analog versions use analogRead(). Adjust the sensitivity screw.
