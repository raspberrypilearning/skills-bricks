UART

Connect Serial

What does it do?

UART is serial communication — sending text and data one bit at a time.

The Arduino uses it to talk to your computer and to modules like GPS or Bluetooth.

With an Arduino, the Serial library sends and receives over USB or pins 0 and 1.

Code

Arduino

void setup() {
  Serial.begin(9600);
}

void loop() {
  Serial.println("Hello");
  if (Serial.available()) {
    char c = Serial.read();
    Serial.println(c);
  }
}
Serial.begin() sets the speed. TX (pin 1) sends and RX (pin 0) receives — these are also used by the USB connection.
