Radio

Connect Receive

What does it do?

A radio module lets two or more Arduinos talk to each other wirelessly.

Modules on the same channel and address can send and receive messages.

With an Arduino, the RF24 library sends and receives over SPI.

Code

Arduino

#include <SPI.h>
#include <RF24.h>

RF24 radio(7, 8);
const byte address[6] = "00001";

void setup() {
  radio.begin();
  radio.openWritingPipe(address);
  radio.stopListening();
}

void loop() {
  const char text[] = "hello";
  radio.write(&text, sizeof(text));
}
Install the RF24 library. The module uses SPI plus CE and CSN pins, and must run on 3.3V.
