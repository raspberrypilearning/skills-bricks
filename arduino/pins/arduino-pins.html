Arduino Pins

Connect Reference

What does it do?

The Arduino's pins are how it connects to the outside world.

Digital pins read or write HIGH and LOW, analog pins (A0–A5) read voltages, and PWM pins (marked ~) can fake an analog output.

Power pins give 5V, 3.3V and GND for your components.

Code

Arduino

const int digitalPin = 2;
const int analogPin = A0;
const int pwmPin = 9;

void setup() {
  pinMode(digitalPin, INPUT);
  pinMode(pwmPin, OUTPUT);
}

void loop() {
  int sensor = analogRead(analogPin);
  analogWrite(pwmPin, sensor / 4);
}
Digital pins use digitalRead/Write, analog pins use analogRead (0–1023), and ~ pins use analogWrite (0–255).
