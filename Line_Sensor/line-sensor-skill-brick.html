Line Sensor

Detect Line

What does it do?

A line sensor helps your project notice dark and light marks underneath it.

It shines infrared light down and checks how much bounces back from the surface.

Your projects can use line sensors for robots, edge finding, counters, and secret floor maps.

Code

Raspberry Pi

from gpiozero import LineSensor

sensor = LineSensor(17)

line_value = sensor.value

Raspberry Pi Pico

from picozero import DigitalInputDevice

sensor = DigitalInputDevice(2)

line_seen = sensor.is_active
Use line_value or line_seen later in your Decide code.
