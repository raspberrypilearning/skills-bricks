Temp Sensor

Detect Heat

What does it do?

A temperature sensor helps your project notice how warm or cold something is.

Many temperature sensors send a changing voltage or digital reading that code can turn into a temperature.

Your projects can use temperature to warn, log, cool, heat, or complain about soup.

Code

Raspberry Pi

from gpiozero import MCP3008

sensor = MCP3008(channel=0)

reading = sensor.value

Raspberry Pi Pico

from picozero import TemperatureSensor

def tmp36(voltage):
    return (voltage - 0.5) * 100

sensor = TemperatureSensor(26, conversion=tmp36)

temp = sensor.temp
Use reading or temp when your project needs to react to temperature.
